NEXA API

High-performance blockchain data API with caching.

API Information

This API provides access to Nexa blockchain data through optimized Electrum server connections utilizing Rostrum protocol methods. We acknowledge and sincerely appreciate the operators of public Rostrum servers, whose ongoing maintenance and contributions play an essential role in enabling this service. Users are kindly reminded to access this service responsibly, ensuring that their usage does not place undue load on public infrastructure or negatively impact the broader community.

  • Intelligent caching for improved performance
  • Request deduplication to reduce server load
  • Automatic failover across multiple servers

Address Endpoints

GET
/balance/:address
Get NEXA balance for an address
Params: address (string)
GET
/transactions/:address
Get transaction history for an address
Query: limit, offset
GET
/utxos/:address
Get unspent transaction outputs
Params: address (string)

Token Endpoints

GET
/tokenbalance/:address
Get token balance for an address
Params: address (string)
GET
/tokentransactions/:address
Get token transaction history
Query: limit, offset
GET
/genesis/:group
Get token genesis information
Params: group (string)
GET
/nftlist/:group
Get list of NFTs in a group
Params: group (string)

Transaction Endpoints

GET
/tx/:txhash
Get decoded transaction data
Params: txhash (hex string)
GET
/rawtx/:txhash
Get raw transaction hex
Params: txhash (hex string)
POST
/broadcast/:tx
Broadcast a signed transaction
Params: tx (hex string)

Blockchain Endpoints

GET
/headers
Get current block tip header with hash
Params: none
GET
/header/verbose
Get verbose tip header with txcount and size
Params: none
GET
/header/verbose/:heightOrHash
Get verbose header by height or block hash
Params: height (integer) or tx (hex string)
GET
/block/:height
Get full block data at height
Params: height (integer)
GET
/mempool
Get mempool transaction count
Params: none

Advanced Endpoints

GET
/metrics
Get API performance metrics
Params: none
